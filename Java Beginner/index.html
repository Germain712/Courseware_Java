Overview

Java is a general purpose, class based, object orientated programming language. It was released in 1995, and is currently one of the most popular programming languages in use.

The language is intended to be a write once, run anywhere language. This means once compiled (i.e. translated in a method that the computer can understand), Java code can run on all platforms that support Java without the need for recompilation.

Java applications achieve this by compiling into byte code that can run on any Java Virtual Machine (JVM), regardless of the underlying computer architecture.

Java has its own syntax that defines the structure of how we can code as well as the phrases we can use. This syntax is the definition of the instructions that the JVM needs to execute.

Why use Java?

There are a number of reasons to use Java as a programming language for a piece of software you are creating. Some of these include:

  • Open-source: the standard edition of Java (Java SE) is free to use!
  • Popularity: Java still remains as one of the most popular languages in the industry, used by huge corporations worldwide.
  • Developement power: the security, complexity, and structure Java provides as a programming language is incredibly useful for projects of all sizes.
